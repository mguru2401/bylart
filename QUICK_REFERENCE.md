## 🚀 Deploy in 3 Steps

### Step 1: Build
```bash
npm run build
```

### Step 2: Deploy to Vercel
1. Go to vercel.com
2. Sign up with GitHub
3. Import your project
4. Click "Deploy"

### Step 3: After Deploy
1. Add domain: bylart.com
2. Submit sitemap to Google Search Console
3. Set up Google Analytics
